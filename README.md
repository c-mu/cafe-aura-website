# ☕ AURA - 架空のカフェブランドサイト

## 🔍 概要
Web制作の基礎である **HTMLとCSS** のスキルを使って制作した、架空のカフェ「AURA」のWebサイトのソースコードです。

* **ターゲット**：20代後半～40代の、落ち着いた空間を求めるビジネスパーソン。
* **目的**：ブランドイメージを確立し、店舗情報へのアクセスを容易にすること。

## 🌐 サイト構成とデザインコンセプト
* **デザイン**：黒、白、ゴールドを使用し、高級感と落ち着きのあるモダンな雰囲気を表現。
* **レスポンシブ対応**：スマートフォン、タブレット、PCの全デバイスで最適なレイアウトを実現。

## 📎 使用技術
* **HTML5**
* **CSS3**（レスポンシブデザイン / CSS変数）
